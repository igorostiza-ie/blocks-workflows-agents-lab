# Blocks, Workflows, and Agents

A comprehensive guide to building effective agentic systems with LLMs.

## About

This lab teaches you how to:
- Understand the difference between workflows and agents
- Choose the right pattern for your use case
- Build with simple, composable patterns
- Implement agents, orchestrators, and evaluators
- Scale agentic systems in production

## Quick Start

1. Visit: https://inakigorostiza.github.io/blocks-workflows-agents-lab/
2. Read through the core patterns section
3. Study the workflow and agent examples
4. Understand when to use each approach

## Prerequisites

- Basic understanding of LLMs
- Familiarity with APIs or Claude Code (optional)

## Duration

- Reading: 30-40 minutes
- Deep dive: 1-2 hours

## Tech Stack

- HTML5, CSS3
- Design System: Inter, Instrument Serif, SF Mono

## Key Topics

- Augmented LLMs (building blocks)
- Prompt chaining workflow
- Routing workflow
- Parallelization workflow
- Orchestrator-workers pattern
- Evaluator-optimizer pattern
- Autonomous agents
- When to use agents vs workflows
- Tool design best practices
